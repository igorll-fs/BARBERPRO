# Wireframes (Descrição / ASCII)

## Login/Cadastro
- Seleção de papel (Cliente / Dono / Funcionário)
- Cliente: telefone (WhatsApp), enviar código, input código 6 dígitos
- Dono: email/senha + Google/Apple

## Cliente - Home
- Carrossel de serviços
- Lista de agendamentos futuros
- Mapa (exibido fora de links públicos)
- Progresso fidelidade
- Stories

## Tela de Agendamento
- Selecionar serviço, barbeiro, data
- Slots de 30/60min
- Aviso se bloqueado

## Funcionário
- Calendário pessoal
- Agendamentos do dia
- Registrar serviço concluído
- Chat com cliente

## Dono - Dashboard
- Gráficos financeiros, filtros
- Estoque (alertas baixo)
- Equipe (criar até 20 contas)
- Promoções e marketing
- Clientes (histórico, bloqueio)

## Perfil Público
- Logo, fotos, serviços, agendamento
- Stories/promos
- Links para baixar o app
